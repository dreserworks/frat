# FRAT — Consolidado de imágenes del sitio

Guía de **todas** las imágenes a entregar. Coloca cada archivo en la carpeta indicada
con el nombre exacto y el sitio las tomará automáticamente.

- Formato recomendado: **.webp** (alternativa .jpg). Logos de marca: **.png** transparente.
- Estética: monocroma / alto contraste / fotografía editorial premium (B/N o desaturada).
- Las dimensiones son **mínimos sugeridos** (×2 para retina). Respetar la **proporción**.

Leyenda de proporción: V = vertical (portrait) · H = horizontal (landscape) · C = cuadrada.
